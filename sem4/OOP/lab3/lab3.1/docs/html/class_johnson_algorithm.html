Detailed Description

Implements Johnson's algorithm for finding all-pairs shortest paths.

Johnson's algorithm combines Bellman-Ford and Dijkstra algorithms to efficiently compute shortest paths between all pairs of vertices in a sparse graph. Supports both sequential and parallel execution.
