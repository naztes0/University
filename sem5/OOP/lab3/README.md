# Lab 3

### Завдання
- Створити програму під Android

> Створити гру морський бій для мобільного телефону (грають комп'ютер та людина).


## Опис

Реалізовано класичну гру "Морський бій" для Android. Гравець грає проти комп'ютера на двох ігрових полях розміром 10x10 клітинок.

## Реалізовані функції

### Основна механіка гри

- Розстановка кораблів на початку гри
- Два ігрові поля: поле гравця та поле противника
- Набір кораблів:
  - 1 чотирипалубний корабель
  - 2 трипалубних кораблі
  - 3 двопалубних кораблі
  - 4 однопалубних кораблі

### Режими складності

Реалізовано два рівні складності:

**Простий режим**
- Комп'ютер стріляє випадковим чином по полю гравця

**Складний режим**
- Комп'ютер використовує розумну стратегію
- Після влучання добиває корабель, стріляючи по сусідніх клітинках
- Визначає напрямок корабля та продовжує атаку в правильному напрямку

### Візуальні елементи

- Різні кольори для різних станів клітинок:
  - Вода (світло-синій)
  - Корабель гравця (сірий)
  - Влучання (червоний хрестик)
  - Промах (світло-сірий)
  - Потоплений корабель (темно-сірий)
- Діалогові вікна для початку гри та закінчення




## Технічні деталі

### Структура проекту

**Пакет game**
- `Board.java` - логіка ігрового поля, розміщення кораблів, обробка пострілів

**Пакет objects**
- `Cell.java` - клас для представлення однієї клітинки поля
- `Ship.java` - клас для представлення корабля

**Пакет ui**
- `MainActivity.java` - головна активність з логікою інтерфейсу та управлінням грою
